# HomestayDorm

This is a [Next.js](https://nextjs.org) project initialized with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Prerequisites

- **Node.js**: 18.17.0 or later
- **npm**: Next.js uses npm by default for this setup

## Installation

1. Clone the repository and navigate into the project directory.
2. Install the necessary dependencies:

   ```bash
   npm install
   ```

## Development

First, run the development server. We use Turbopack for faster local development:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Available Scripts

In the project directory, you can run:

- `npm run dev` - Starts the development server with Turbopack enabled (`next dev --turbo`).
- `npm run build` - Builds the application for production usage.
- `npm run start` - Runs the compiled application in production mode.
- `npm run lint` - Runs ESLint to find and fix any linting errors in the codebase.
